Preguntas fecuentes

¿Cómo me puedo registrar y Logear?

Como primer paso, debe de presionar el texto que dice "login" en la esquina superior derecha. Este lo llevará a una ventana con un boton llamado "Opciones". Si usted coloca el mouse sobre este botón, se mostrará el boton de "registro" y de "login". Seleccione la opcion que desea y llene el formulario.

¿Qué significa cada botón en el Módulo de comentarios?

El botón verde es para ir directamente al comentario reportado, el botón rojo es para ignorar el comentario, y el botón naranja es para poder censurar el comentario.

¿Qué significa cada botón en el Módulo de Artículos ?

El botón naranja es para poder visualizar el artículo en cola, el botón verde es para aprobar el artículo, y el botón rojo es para enviar un correo.
